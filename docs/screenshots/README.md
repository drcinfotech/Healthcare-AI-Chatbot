## Capturing the app

1. Start both backend and frontend:
   ```bash
   docker compose up
   ```
2. Open <http://localhost:5173> in Chrome at 1400×900 viewport
3. Take screenshots (Cmd/Ctrl + Shift + 4 on macOS, or Snipping Tool on Windows)
4. Drop them in this folder using the filenames above
5. They'll auto-render in the main README's Preview section
